# Security Policy

## Supported Versions

Only the latest version of DevQuest receives security updates. If a vulnerability is found, please report it immediately as outlined below.

## Reporting a Vulnerability

We take the security of DevQuest seriously. If you find any security vulnerability, please do **not** open a public issue. Instead, please report it directly by contacting the maintainer via GitHub profile contact details.

Once a report is received, we will investigate and address the issue promptly.
